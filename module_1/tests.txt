Пользовательский сценарий
2.1 Редактирование пользователя
Тестовый сценарий:
2.1.1 Успешное изменение пароля пользователя
Предусловие: Пользователь предварительно зарегистрирован в системе и залогинен. Открыта вкладка "Профиль".
Тестовые данные: 
Логин для входа: hello1234@mail.ru
Пароль для входа (старый пароль): Hello123456
Новый пароль: Hello789123
Шаги воспроизведения:
1. Нажать "Изменить пароль". Ожидаемый результат: Отображение блока "Изменить пароль" с полями "Старый пароль", "Новый пароль" (с валидирующими сообщениями), "Подтверждение нового пароля", кнопа "Сохранить", ссылка "отмена".
2. Ввести Пароль для входа в поле "Старый пароль". Ожидаемый результат: Поле "Старый пароль" заполнено. Значение отображается валидирующими символами
3. Ввести Новый пароль в поле "Новый пароль". Ожидаемый результат: Поле "Новый пароль" заполнено. Значение отображается валидирующими символами
3. Ввести Новый пароль в поле "Подтверждение нового пароля". Ожидаемый результат: Поле "Подтверждение нового пароля" заполнено. Значение отображается валидирующими символами  
4. Кликнуть "Сохранить". Ожидаемый результат: Отображение блока "Профиль". Успешное сообщение "Пароль обновлён".

2.2 Удаление пользователя
Тестовый сценарий:
2.2.1 Успешное удаление пользователя
Предусловие: Пользователь предварительно зарегистрирован в системе и залогинен. Открыта вкладка "Профиль".
Тестовые данные: 
Логин для входа: Электронная почта: можно получить на https://www.temporary-mail.net/ 
Пароль для входа: Hello789123
Шаги воспроизведения:
1. Нажать "Удалить профиль". Ожидаемый результат: Отображение блока "Удалить профиль", поля "Пароль", текстового блока "Внимание", кнопки "Удалить", ссылки "отмена"
2. Ввести Пароль для входа. Ожидаемый результат: Поле "Пароль" заполнено. Значение отображается символами валидации
3. Кликнуть "Удалить". Ожидаемый результат: Отображение успешного сообщения "Ваш профиль удален. Спасибо, что воспользовались нашим сайтом.". В шапке профиля присутствует ссылка "Войти или зарегистрироваться".
